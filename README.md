# Controle de Suprimentos de Impressão

Sistema web desenvolvido para controle de suprimentos de impressão utilizando HTML, Tailwind CSS, JavaScript e Supabase.

## Funcionalidades

- Cadastro de usuários
- Login e autenticação
- Área restrita
- Cadastro de suprimentos
- Edição de suprimentos
- Exclusão de suprimentos
- Registro de movimentações
- Dashboard com indicadores
- Gráfico de movimentações por destino

## Tecnologias Utilizadas

- HTML5
- Tailwind CSS
- JavaScript
- Supabase
- PostgreSQL
- Chart.js
